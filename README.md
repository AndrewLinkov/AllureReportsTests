## AllureReportsTests

### Описание 
В проекте реализовано применение Allure отчета  

### 1. Чек лист для формирования Allure отчета, необходимо
1. Подключить библиотеки Allure и плагин
2. Настроить allure.properties
3. Алгоритм запуска отчета:
- Запустить тесты
- Allure:report - сформировать отчет
- Allure:serve - отобразить отчет
